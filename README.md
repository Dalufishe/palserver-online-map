# Palserver Map

Online map for the Palworld Dedicated Server, which displays players' locations, levels, and names.

![](./preview.png)

### How to use

#### Run server

```bash
npm i
```

```bash
node server.js
```

#### Start using the Map

1. Click connect to server button.
2. Enter your Ip address, rest api port and admin password.
3. Have fun!
